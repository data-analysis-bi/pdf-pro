# Contributing to PDFPro

Thank you for your interest in contributing to PDFPro!  
We welcome improvements, bug fixes, new features, and documentation updates.

---

## 🧩 How to Contribute

### 1. Fork the repository
Click the "Fork" button on GitHub.

### 2. Clone your fork
'''
git clone https://github.com/<your-username>/pdf-pro.git
'''
### 3. Create a feature branch
git checkout -b feature/my-new-feature

### 4. Install dependencies
npm install

### 5. Make your changes
Follow the existing code style and structure.

### 6. Run the dev server
npm run dev

### 7. Commit your changes
git commit -m "Add: description of your change"

### 8. Push your branch
git push origin feature/my-new-feature

### 9. Open a Pull Request
Describe your changes clearly and link any related issues.

---

## 🧪 Code Style

- Use React functional components
- Prefer TailwindCSS utility classes
- Keep components small and modular
- Avoid unnecessary dependencies

---

## 🐛 Reporting Issues

If you find a bug:
- Open an issue
- Describe the problem
- Include steps to reproduce
- Add screenshots if possible

---

## ❤️ Thank You

Your contributions help make PDFPro better for everyone!
